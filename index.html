What is Perl?

Perl is a high-level, general-purpose scripting language traditionally used for text processing, system administration, web development, and automation tasks. It supports multiple paradigms, including procedural, object-oriented, and functional programming.

  • High-level interpreted scripting language
  • Excellent for text parsing and regular expressions
  • Strong CPAN library ecosystem
  • Cross-platform and portable

Language History

Perl was created by Larry Wall in 1987 while he was working as a programmer for NASA. It was intended to make report processing and shell-style scripting easier. Perl 5 was released in 1994 and remains the dominant version today, with Perl 7 in development as the future modernization path.

Example Programs

Hello World

#!/usr/bin/perl
print "Hello World!\n";

Run:

perl hello.pl

Variables and Input Example

#!/usr/bin/perl
print "Enter your name: ";
$name = <STDIN>;
chomp($name);
print "Hello, $name!\n";

Loop Example

#!/usr/bin/perl
for ($i = 1; $i <= 5; $i++) {
    print "Count: $i\n";
}

Install & Running Perl

Check if Perl is Installed

perl -v

Install (Linux / macOS)

sudo apt install perl
brew install perl

Windows

Download from Strawberry Perl: https://strawberryperl.com

Running a Script

perl program.pl
